Introduction to Vehicle Forensics

This guide will introduce you to the fundamental tools and techniques used in forensic vehicle analysis.

What is Vehicle Forensics?

Vehicle forensics is the practice of applying digital forensic techniques to vehicles. This allows investigators to uncover useful information which can help in their investigation

What data can be found in a vehicle?

  • ✅ Speed, braking and movement data
  • ✅ GPS location data and search history
  • ✅ Black box (EDR) data analysis
  • ✅ Digital car diagnostics
